# Spacing

FDND uses relative spacing based on `em` units, so spacing scales with the font size of the element.

## Typography spacing

The image below shows the spacing relationships between typography elements as defined in the Figma design.

<img src="/spacingfigma.png" alt="Figma spacing voorbeeld foto" />

These rules define how much vertical space sits between two text elements that directly follow each other.
